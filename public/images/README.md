# Images Directory

Place your storyboard images here.

## Naming Convention (suggested)

- `sound-vibe-1.jpg` - SOUND & VIBE section, first image
- `sound-vibe-2.jpg` - SOUND & VIBE section, second image
- `visual-imagery-1.jpg` - VISUAL & IMAGERY section, first image
- `visual-imagery-2.jpg` - VISUAL & IMAGERY section, second image
- `production-1.jpg` - PRODUCTION NOTES section, first image
- `production-2.jpg` - PRODUCTION NOTES section, second image
- `lyrics-1.jpg` - LYRICS section, first image
- `lyrics-2.jpg` - LYRICS section, second image
- `vibe-theme-1.jpg` - THEME & EMOTIONAL ARC section, first image
- `vibe-theme-2.jpg` - THEME & EMOTIONAL ARC section, second image

## Supported Formats
- .jpg / .jpeg
- .png
- .gif
- .webp

## After Adding Images
Update the image paths in `src/data/wantsAndNeeds.ts` to use `/images/your-filename.jpg`
